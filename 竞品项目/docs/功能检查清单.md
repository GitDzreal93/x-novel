## 📦 项目管理模块

### 创建项目
- [ ] 项目名称输入（最大 50 字符，带计数）
- [ ] 小说主题/核心创意输入（最大 500 字符，带计数）
- [ ] 小说类型多选（15 种类型）
- [ ] 预计章节数（10-500，默认 100）
- [ ] 每章字数（1000-10000，默认 3000）
